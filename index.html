Integração com Robôs / Ferramentas Externas

Gere o seu link exclusivo de Webhook para receber contatos (leads) automaticamente no seu Funil de Vendas, vindo de ferramentas como BotConversa, Make, N8N, ChatBot, Zapier, etc.

Em sua aplicação de Origem (Ex: BotConversa), acesse as Integrações/Webhooks e cole este link apontando o método para POST. O sistema de triagem (Handoff) direcionará automaticamente as mensagens para cá.
